DrawingAnalysisMiya

2Dの機械図面（PDF形式または画像形式）をAIが解析し、図面に描かれている各部品の寸法を読み取って自動的に体積を割り出すデスクトップアプリケーションです。

アプリの概要と機能

PDF/画像対応: PDFの図面ファイルや、PNG/JPEGなどの画像ファイルを直接読み込むことができます。

高度な画像前処理: 図面のかすれやノイズを自動で補正し、AIが寸法を読み取りやすい状態に最適化します。

AIによる自動計算: 最新のAIモデル（Gemini API）を活用し、図面から部品名と寸法を抽出し、体積（mm³）を自動計算して一覧表示します。

動作環境

Windows 10 または Windows 11 (64bit)

インターネット接続環境（AI解析に必須）

Gemini API キー（Google AI Studioから無料で取得可能です）

インストール・実行方法

配布された DrawingAnalysisMiya.exe などの実行ファイルを任意のフォルダに保存します。

ファイルをダブルクリックして起動します。

⚠️ 初回起動時の警告画面（SmartScreen）について

Windowsのセキュリティ機能（Windows Defender SmartScreen）により、初回実行時に**「Windows によって PC が保護されました」**という青い警告画面が表示される場合があります。

これはアプリの不具合やウイルスではなく、新しく作成されたアプリによく表示される標準的な確認画面です。以下の手順で起動してください。

警告画面内の 「詳細情報」 という文字をクリックします。

右下に現れる 「実行」 ボタンをクリックします。
（次回以降はこの画面は表示されず、すぐに起動するようになります）

基本的な使い方

アプリを起動したら、画面左上の「Gemini API キー」欄にご自身のAPIキーを入力します。

「ファイルを選択」ボタンをクリックし、解析したい2D機械図面のファイル（PDFまたは画像）を選びます。

「解析を実行」ボタンをクリックします。

解析が開始されます。図面が最適化されたプレビューが表示され、数十秒待つと右側の表に部品ごとの寸法と体積の計算結果が表示されます。

※注意事項: AIによる読み取りや計算結果は推測を含む場合があるため、実作業の際は必ず目視での確認・検証を行ってください。
